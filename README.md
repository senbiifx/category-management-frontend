# Category Management (Frontend)

This project covers the following functionalities:
  - Displaying the categories in Tree View
  - Adding categories
  - Updating categories
  - Deleting categories

![Alt text](demo.gif?raw=true "Demo")


## How to Run

The pre-requisites are:
  - You have set up the Angular CLI
  - You have set up the back-end server. https://github.com/senebii/category-management-backend

Go the directory then run `ng serve`. Navigate to `http://localhost:4200/`.
